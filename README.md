# Python 교육 과정

Python 기초부터 중급까지 단계별로 학습하는 과정입니다.  
파이썬 기초 문법과 객체 지향 프로그래밍을 배우고,  
alphagenome API 사용법까지 익힐 수 있습니다.

## 과정 구성

### Basics (기초 과정)
**디렉토리**: `Basics/`

Python 프로그래밍의 기초를 다룹니다:
- **Kbiohealth-python.ipynb**: 종합적인 Python 기초 학습 노트북
- **calculation.py**: 계산 관련 기본 함수와 예제
- **calculation_practice.py**: 계산 실습 문제
- **christmas.py**: 크리스마스 테마 프로그래밍 예제
- **dinner_menu_simple.py**: 간단한 메뉴 관리 프로그램
- **dinner_menu_detail.py**: 상세한 메뉴 관리 프로그램

### Intermediate (중급 과정)
**디렉토리**: `Intermediate/`

Python의 중급 개념과 실제 프로젝트를 다룹니다:
- **1_class_practice.ipynb**: 클래스와 객체지향 프로그래밍 실습
- **quick_start.ipynb**: 알파 genome quick start 튜토리얼
